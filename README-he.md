# HighwayOne

כלי פשוט ליצירת שלטי כבישי ישראלי.  
[הפעל את HighwayOne](https://omgaler.github.io/HighwayOne/)

## סקירה כללית

HighwayOne הוא כלי קל מינימלי ליצירת שילוט כבישים ישראליים ישירות בדפדפן. ניתן להזין מספר וסוג כביש ולקבל שלט מעוצב בהתאם לשיטת הצבעים המוכרת של שילוט הדרכים בישראל.

האפליקציה:

- מהירה וללא תלות בספריות כבדות  
- מדויקת ויזואלית במידה מספקת עבור מוקאפים ומצגות  

## תכונות

- יצירת שילוט לפי מספר  
- בחירת צבע אוטומטית לפי סוג הדרך:
  - **כחול** – כבישים מהירים  
  - **אדום** – כבישים ארציים  
  - **ירוק** – כבישים אזוריים  
  - **שחור** – כבישים מקומיים  
- רינדור נקי באמצעות HTML/CSS/SVG  
- תצוגה מיידית (ללא רענון)  
- אפליקציה סטטית  

## איך זה עובד

1. בחר סוג כביש  
2. הזן מספר כביש  
3. שילוט מעוצב נוצר באופן דינמי  
4. התוצאה מוצגת מיידית וניתנת להעתקה או להורדה כקובץ תמונה (PNG) או וקטורי (SVG)
